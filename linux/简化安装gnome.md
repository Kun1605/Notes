1. ubuntu最简化安装使用gnome桌面             

   ​             1、gnome桌面窗口管理程序

   ​             

   ```
   $sudo apt-get  install gnome-shell  
   ```

    

2. ​           2、安装gnome面板

   ​         

   ```
     $sudo apt-get  install  gnome-panel  
   ```

    

3. ​           3、安装gnome菜单

   ​         

   ```
     $sudo apt-get  install   gnome-menus
   ```

    

4. ​          4、安装gnome-session

   ​         

   ```
      $sudo apt-get  install  gnome-session
   ```

    

5. ​          5、安装gdm会话切换器

   ​        

   ```
        $sudo apt-get  install  gdm
   ```
